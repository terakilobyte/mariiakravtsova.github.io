Tutorials Point

Tutorials Point provides a very easy to follow set of guides for any person looking to use HTML. Each section provides examples with a built-in preview of what your code does. Having finished all the guides you will know how to properly code using Html.

HTML Dog

HTML dog provides guides for both HTML and CSS with different courses ranging from beginners to advanced problems. They also provide examples to showcase what they are teaching for any level.

Free Code Camp

Begin your quest with HTML, CSS, then follow it all the way to becoming a full stack web developer. Most of the tutorials are in browser based. Also, a great and supporting community.

Udemy

Learn by watching the videos and following along. You will build websites that can be used for your portfolio. It gives you just enough surface information, so you can start building things.

CodeCademy

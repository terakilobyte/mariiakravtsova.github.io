Predator - Prey Simmulation

The Lotka Volterra equations, also known as the predator prey equations, are a pair of first order, non linear, differential equations frequently used to describe the dynamics of biological systems in which two species interact, one as a predator and the other as prey. The populations change through time according to the pair of equations. Read Wikipedia to learn more.


This is an implementation of Lotka Volterra equations using JavaScript and jQuery. Completed by Mariia, Luke, and Justin for a CS 3500 class. We used multiple resources to collect the code, check comments to see them.
